Selector Cheatsheet

by Joun Obey

Selector Name Selector Description
p Type Selector CSS type selectors are used to match all elements of a given type or tag name
. Class Selector The CSS class selector matches elements based on the contents of their class attribute. For selecting elements having calendar-cell as the value of the class attribute, a . needs to be prepended.
# ID Selectors CSS ID selectors should be unique and used to style only a single element. ID selectors are denoted with a hash sign # followed by the id name.
